# 🚀 Devendiran T - Professional AI & ML Portfolio

A clean, modern, professional portfolio website showcasing AI/ML projects, experience, and skills. Built with React, Tailwind CSS, and designed for software engineering and AI/ML job applications.

## 🎨 Design System

### Color Palette:
- **Primary Blue:** `#2563EB` - CTAs, links, accents
- **Secondary Blue:** `#0EA5E9` - Gradients, hover states
- **Background:** `#FFFFFF` - Main background
- **Light Gray:** `#F8FAFC` - Alternate sections
- **Text:** `#1F2937` - Headings and body text

### Typography:
- **Headings:** Poppins (Bold, Black weights)
- **Body:** Inter (Regular, Medium weights)
- **Clean spacing and excellent readability**

### Components:
- **Cards:** 12px border radius, soft shadows
- **Buttons:** 8px radius, smooth transitions
- **Spacing:** Consistent 80px section padding
