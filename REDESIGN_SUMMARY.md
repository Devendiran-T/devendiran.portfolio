## 📝 Customization Tips

### Change Colors:
Edit `tailwind.config.js`:
```javascript
colors: {
  primary: "#YOUR_COLOR",
  secondary: "#YOUR_COLOR",
}
```

### Change Fonts:
Edit `tailwind.config.js` and `src/index.css`:
```javascript
fontFamily: {
  inter: ['YourFont', 'sans-serif'],
}
```

### Adjust Spacing:
Edit spacing values in `tailwind.config.js`:
```javascript
spacing: {
  section: "100px", // Change section padding
}
```
