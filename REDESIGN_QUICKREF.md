## 🔧 Customize Your Portfolio

### 1. Update Content
Edit `src/constants/index.js`:
- Personal info
- Projects
- Experience
- Skills

### 2. Change Colors
Edit `tailwind.config.js`:
```javascript
colors: {
  primary: "#YOUR_COLOR"
}
```

### 3. Change Fonts
Edit `src/index.css`:
```css
@import url('YOUR_GOOGLE_FONTS_URL');
```
