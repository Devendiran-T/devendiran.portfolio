# ⚡ Quick Start Guide

## 🎯 Fastest Way to Run

### Method 1: Double-click the startup script
```
start.bat
```
Just double-click `start.bat` in Windows Explorer!

### Method 2: PowerShell Command
```powershell
cd "c:\Users\Mani\Downloads\devendiran.portfolio-main\devendiran.portfolio-main"
npm run dev
```

### Method 3: Run Both Frontend + Backend
```
start-both.bat
```
Double-click `start-both.bat` to launch both servers!

---

## 🌐 Access Your Portfolio

Once running, open your browser to:
**http://localhost:5173**

---

## 🛑 Stop the Server

Press `Ctrl + C` in the terminal window

---

## 📝 First Time Setup Only

If this is your first time running the project:

```powershell
npm install
```

That's it! The project is already configured and ready to run.
