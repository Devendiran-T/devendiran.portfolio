# Profile Photo Instructions

## How to Add Your Photo

1. Save your professional photo as `profile.jpg` in this `public` folder
2. The photo will be displayed as a circular image in the hero section
3. Recommended specifications:
   - Format: JPG or PNG
   - Size: 800x800 pixels (square)
   - File name: `profile.jpg`

## Photo Guidelines

**Professional Photo Tips:**
- Clean, professional background (white or neutral)
- Business attire (suit/formal wear)
- Good lighting
- Centered face and upper body
- High resolution (at least 800x800px)
- Friendly, professional expression

**Technical Requirements:**
- The image will be cropped to a perfect circle
- White border will be added automatically
- Image should be square (1:1 aspect ratio)
- File should be optimized (under 500KB for web)

## Fallback

If no photo is provided, the system will show your initial letter (D) in a blue circle as a placeholder.

## File Location

Place your photo here:
```
public/profile.jpg
```

The portfolio will automatically load it from `/profile.jpg`
